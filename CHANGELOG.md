# Changelog

All notable changes to this project will be documented in this file.

This project follows **0.x preview** semantics: breaking changes may occur between minor versions.

## 0.1.0

- Initial public preview release.
- Coroutine-based Redis client API (`client`, `config`, `request`, typed `response`).
- RESP3 incremental parser and encoder utilities.
- Strand-serialized connection actor with pipelining, timeouts, and reconnection policy.
